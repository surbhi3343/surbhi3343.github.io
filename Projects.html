MY PORTFOLIO PROJECT

Project Title:

Portfolio website designing.

Overview:

A collection of my skills, hobbies, education, accomplishments and contact details.

Objective:

It is a good way of self-discovery and self-confidence.It helps me to organize my accomplishments,goals,and personal thought.

Phases of work:

  • For, my first welcome page I have used a professional background image from chrome and downloaded it in jpg format. I have created a vertical navigation bar using HTML and also the social meadia clickable buttons.And,then I have included CSS for making it more attractive.
    For text animation I have used CSS -webkit-animation property and @keyframes for changing the one set of CSS to another.
    For making the navigation bar more attarctive I have added the relative icons corressponding to the nav bar by including font awesome using JavaScript.
  • Moving to my Home page, I have used a photo of mine as background and put a gray shade using CSS linear gradient property over the background for making it look attractive.For changing the font and shade of text I have used the CSS font-family and text-color properties.
  • In my About page I have included the some pictures in a row format using the img tag in HTML.I have choosed a gray shade as background-color.
    In skills section I have used bars to show my level of skills using box property in CSS.
  • In the Resume Page divided the sections using a hr tag in HTML and included the certifications and used text shadow property in CSS.
  • In the Project I have used the text shadow property in CSS.
  • In the Contact Page attached a google map of my location and created a contact HTML form.
